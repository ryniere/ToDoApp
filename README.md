# Getting Started

## MVVM Architecture


The ToDo app was developed using MVVM architectecture and RxSwift.



# Running

Go to the project folder in the terminal and run:

```
pod install
```

* Open the ToDoApp.xcworkspace
* Select an iOS simulator
* Press play to run the app


## Features

- [x] Fetch Data
- [x] Completed Filter
- [x] Search Text
- [x] Accordion
